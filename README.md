# Chattering

## Requirements
- Docker
- Docker Compose
- Node.js
- npm

## Installation and Running

### 1. Run MySQL Database

```sh
docker-compose up -d
```

### 2. Install and Run React Application

```sh
npm install
npm run build
npm start
```

http://localhost:3000
